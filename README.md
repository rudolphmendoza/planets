# Planets Website

## Overview

The Planets Website is a simple educational website that provides information about planets in our solar system. The goal of this project is to practice web development skills using HTML and CSS while creating an attractive and informative webpage.

## Features

* Displays information about planets
* Includes images of planets
* Custom styling with CSS
* Easy-to-navigate layout
* Educational content about the solar system

## Technologies Used

* HTML
* CSS
* Visual Studio Code
* Git
* GitHub

## Project Structure

```text
planets-project/
│
├── planets.html
├── style.css
├── README.md
└── images/
```

### File Descriptions

* **planets.html** - Main webpage containing the content and structure.
* **style.css** - Stylesheet used to control the appearance of the website.
* **README.md** - Documentation for the project.
* **images/** - Contains planet and background images used by the website.

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `planets.html` in a web browser.

Or, if Live Server is installed:

1. Right-click `planets.html`.
2. Select **Open with Live Server**.

## Git and GitHub

Git was used to track changes throughout development. GitHub was used to store the repository online and manage project versions.

## Security

This project does not require user accounts, passwords, or database connections. Any unnecessary files are excluded from the repository using `.gitignore` when appropriate.
